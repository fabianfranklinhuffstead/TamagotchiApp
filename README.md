# Tamagotchi App

<p>A basic pure C# solution for creating 1996 <a href="https://en.wikipedia.org/wiki/Tamagotchi">Tamagotchi  Japanese Toy</a></p>

<h2>The task: </h2>
<ul>
  <li>Initialize game</li>
  <li>Allow user to change name and save</li>
  <li>Feed Tamogochi</li>
  <li>Play with Tamogochi</li>
  <li>Talk Tamogochi</li>
  <li>Heal Tamogochi</li>
  <li>End game</li>
</ul>
